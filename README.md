# OTUS 30-min lesson project

Это учебный проект для практической работы по профилированию 
на 30-минутном демо занятии в OTUS.

## Что делает проект
Парсит CSV файл с тестовыми контактами, валидирует их и в зависимости от успеха валидации раскладывает по файлам с валидными и невалидными результатами

## Цели практики
* Получить опыт запуска Xhprof на проекте и умение интерпретации результатов его работы
* С помощью Xhprof найти проблемные места и обозначить направления для улучшения

## Предусловия
* Локально установленный [Docker](https://www.docker.com/)

## Быстрый старт
1. Клонировать репозиторий
2. Выполнить команду `make install`
3. Открыть [http://localhost:8000](http://localhost:8000) в браузере
4. Открыть страницу с результатами профилирования [http://localhost:8000/xhprof_html/](http://localhost:8000/xhprof_html/)
