# OTUS 30-min lesson project

Это небольшой проект для практической работы на 30-минутном демо занятии в OTUS.

## Предусловия
* Локально установленный [Docker](https://www.docker.com/)

## Быстрый старт
1. Клонировать репозиторий
2. Выполнить `docker-compose up -d`
3. Открыть `http://localhost:8000` в браузере
