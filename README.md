# OTUS 30-min lesson project

Это небольшой проект для практической работы на 30-минутном демо занятии в OTUS.

## Цели практики
* Узнать что такое Xhprof и как он встраивается в проект
* Развернуть тестовое приложение
* Снять характеристики работы эндпоинта [http://localhost:8000/performance](http://localhost:8000/performance)
* Посмотреть результаты профилирования по адресу [http://localhost:8000/xhprof_html](http://localhost:8000/xhprof_html)

## Предусловия
* Локально установленный [Docker](https://www.docker.com/)

## Быстрый старт
1. Клонировать репозиторий
2. Выполнить `docker-compose up -d`
3. Открыть `http://localhost:8000` в браузере
